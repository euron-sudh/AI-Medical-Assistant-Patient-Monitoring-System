Project Overview

MedAssist AI is a full-stack, agentic AI-powered medical assistant and patient monitoring platform designed for hospitals, telemedicine providers, and health startups. It combines symptom analysis, medical report interpretation, continuous patient monitoring, voice-based interaction, and intelligent alerting into a single unified, HIPAA-compliant system.

The platform leverages seven specialized AI agents coordinated by a central orchestrator, each powered by OpenAI GPT-4o with function calling, retrieval-augmented generation, and medical knowledge grounding to deliver accurate, context-aware clinical insights to both patients and healthcare professionals.

Problem Statement

🚨

Delayed Diagnoses

Patients wait days or weeks for specialist appointments while conditions worsen. Symptom triage is manual and inconsistent across providers.

📄

Opaque Medical Reports

Lab results and imaging reports are written in medical jargon that patients cannot understand, creating anxiety and confusion about their health status.

💊

Medication Errors

Drug interactions and dosage errors cause 7,000+ deaths annually in the US alone. Manual cross-checking is error-prone and time-consuming.

📈

Reactive Monitoring

Patient vitals are checked periodically, missing critical deterioration windows. Early warning signs go undetected until conditions become emergencies.

📝

Documentation Burden

Physicians spend 2+ hours daily on clinical documentation instead of patient care, leading to burnout and reduced quality of care.

🔒

Compliance Complexity

HIPAA compliance requires comprehensive audit trails, encryption, and access controls that are costly to implement and maintain across disparate systems.

AI Agent System

MedAssist AI is built around seven specialized AI agents coordinated by a central Agent Orchestrator. Each agent extends a common BaseAgent class, uses OpenAI function calling, and accesses data exclusively through a validated tool layer.

1. Symptom Analyst Agent

GPT-4o

Conducts structured multi-turn symptom interviews using the OLDCARTS framework. Produces ranked differential diagnoses with confidence scores, urgency assessments, and specialist referrals. Tools: search_medical_kb, query_patient_history, calculate_urgency_score, recommend_specialist.

2. Report Reader Agent

GPT-4o Vision

Ingests and interprets medical reports (lab results, imaging, pathology) using vision and text analysis. Extracts structured lab values, identifies abnormalities, and generates both clinician-grade summaries and patient-friendly explanations. Tools: parse_lab_values, identify_abnormalities, explain_report, correlate_with_history.

3. Triage Agent

GPT-4o-mini

Performs real-time emergency triage with ESI (Emergency Severity Index) level assignment. Near-instant response (<2 seconds) with conservative bias toward higher urgency. Automatically escalates ESI Level 1-2 to emergency protocols. Tools: calculate_esi_score, check_emergency_symptoms, assign_priority.

4. Voice Agent

Whisper + TTS

Handles all voice-based interactions including speech-to-text transcription (50+ languages via Whisper), AI-powered response generation, and text-to-speech synthesis via EURI TTS API for high-quality audio output audible in video calls. Tools: transcribe_audio, synthesize_speech, generate_clinical_notes.

5. Drug Interaction Agent

GPT-4o-mini

Analyzes medication lists for drug-drug interactions, drug-allergy cross-references, dosage verification, and contraindications. Provides severity classifications with evidence-based citations and recommends safer alternatives. Tools: check_interactions, verify_dosage, suggest_alternatives, check_allergies.

6. Patient Monitoring Agent

GPT-4o-mini

Continuously monitors incoming patient vitals, detects anomalies via statistical + AI methods, calculates NEWS2/MEWS early warning scores, and triggers multi-level alert escalation chains. Predictive alerts 30-60 minutes before clinical deterioration. Tools: read_vitals, calculate_news2_score, detect_anomalies, trigger_alert.

7. Follow-Up & Care Plan Agent

GPT-4o

Generates personalized, evidence-based care plans with measurable goals, schedules follow-ups, sends multi-channel reminders, and tracks treatment adherence. Adapts plans based on patient progress and feedback. Tools: generate_care_plan, track_adherence, adjust_care_plan, schedule_followup.

Security & HIPAA Compliance

ControlImplementation
Encryption at RestAES-256 for all PHI columns in PostgreSQL; S3 server-side encryption
Encryption in TransitTLS 1.3 enforced on all endpoints; HSTS with 1-year max-age

Authorization4-tier RBAC (patient, nurse, doctor, admin); resource-level access control

Input ValidationPydantic schemas on all endpoints; parameterized SQL queries only
